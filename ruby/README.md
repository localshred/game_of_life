## See it in action

Checkout the [video][] I posted showing a simple 435 generation run. It's also
a [rubygem][] if you're into that sort of thing.

## Run the game!

```shell
$ ruby game.rb
```

Or, run the game for a specified board size by providing an integer:

```shell
$ ruby game.rb 50
```
